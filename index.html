2024 Presidential Election Polls

In November, the United States will have its presidential election. In this dashboard, we'll explore the historical polling data for the top three candidates and whether current events, national and international, had an impact on their poll scores.

This dashboard was inspired by FiveThirtyEight's dashboard. The dataset is also from FiveThirtyEight, though I'm using data up to July 20, 2024. The latest version can be downloaded here.

Average Poll Scores of the Top 3 Presidential Candidates

The top candidates who are running for president in 2024 include the following: current president Joseph Biden, former president Donald Trump, and Robert F. Kennedy Jr., the son of Robert F. Kennedy and nephew of former president John F. Kennedy.

More about this dataset

As mentioned earlier, this dataset is taken from FiveThirtyEight. However, this dataset is quite large and includes data for all current and former candidates.

I used a free Azure Data Explorer cluster to explore the dataset; clean and filter to the top 3 candidates and polls only from 2024; and to aggregate multiple daily poll scores into averages. First, I ingested the dataset into my free Azure Data Explorer cluster, adjusting the schema to make sure fields such as datetimes were imported correctly. Then, I ran the following query to select and aggregate the data to what I needed: 

                
                    let data = polls_2024_07_20
                    | where start_date > datetime(2024-01-01) and answer in ('Biden', 'Trump', 'Kennedy');
                    let stats = data | summarize avg(pct), stdev(pct) by answer;
                    data
                    | join kind=leftouter stats on $left.answer == $right.answer
                    | extend min_pct = avg_pct - 1 * stdev_pct, max_pct = avg_pct + 1 * stdev_pct
                    | where pct > min_pct and pct < max_pct
                    | summarize round(avg(pct), 2) by start_date, answer
                    | project Date = start_date, Candidate = answer, AverageScore = avg_pct
From Azure Data Explorer, I was able to export these query results to a csv file, and use that to build my d3 charts.
